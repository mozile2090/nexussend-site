Data Processing Agreement

This DPA supplements the Terms of Service regarding the processing of personal data.

Ephemeral Processing Guarantee

NexusSend acts as a data processor for the payloads you transmit through our routing APIs. We process data entirely in memory (RAM). Payload obfuscation and routing are executed instantly without writing plain-text email content to disk.

OAuth Credential Security

Google API tokens are encrypted at rest using AES-256. They are only decrypted momentarily by our core queue workers when executing an authorized transmission.
